What’s included in this site

This site showcases the Neotropical Plants repository. The repository used to build this site sources code from that repository to generate the corresponding html files. - analysis: sources the R Markdown files for the workflowr website - docs: contains the webpages generated from the R Markdown files from the analysis directory

This project follows the Environmental Data Initative (EDI) workflow. As such, the L0 code retrieves the raw data, the L1 code modifies and cleans the data, and the L2 code runs analyses with the data.
